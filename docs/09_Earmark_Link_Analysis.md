## Key Findings

The earmark link analysis produced the following observations:

- A total of **1,013,414** transaction IDs ending in `E` were identified.
- Every `E` transaction had a corresponding base transaction.
- The linked transactions always reported the **same contribution amount**.
- The linked transactions always belonged to the **same committee**.
- Contributor names were different for all linked pairs, suggesting that the two records represent different reporting perspectives of the same earmarked contribution.
- Contribution dates matched for only **28.97%** of linked pairs, indicating that intermediary organizations and campaign committees may report the transaction on different dates.

### Research Implication

The analysis demonstrates that transactions ending with `E` should not be treated as independent donations. Instead, they represent linked reporting records that must be reconciled before calculating a donor's true contribution amount.
